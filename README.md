# SUBNET LIFT – Vertical Order v2.5

**SUBNET LIFT** is a high-fidelity, architectural elevator logistics simulation set in a corporate dystopia. v2.5 introduces the **"High-Fidelity Verticality"** overhaul, featuring full-scale architectural rooms and advanced spirit animations.

## 🏗️ Architectural Evolution (v2.5)
- **High-Fidelity Rooms**: Every floor is now a **120px architectural room**, replacing the abstract list with an immersive vertical environment.
- **Synchronized Scrolling**: The mega-tower stands at a massive **3600px tall**, navigated via a synchronized vertical scroll that links the floor lobbies with the elevator shafts.
- **Proportional Scaling**: Character sprites have been scaled to **80px**, and elevator cars to **100px**, ensuring a one-to-one visual mapping within the tower's geography.

## 👻 Walking Spirits v2
- **Directional Pathfinding**: Spirits now turn naturally using `scaleX` flipping when reaching room boundaries.
- **Spatial Depth**: Randomized vertical offsets and dynamic Z-index layering prevent visual overlapping and create a lively, non-robotic population.
- **Class-Specific Avatars**: Individualized sprites for **Executives (Cloaked)**, **Workers (Ethereal)**, and **Drones (Service Units)**.

## 🎮 Logistics Operations
- **🕹️ MANUAL_CONTROL**: Refined dispatch logic with input validation. Click elevator cars to set targets (0-29) or use the enlarged tactical **CALL** buttons.
- **💻 LOGISTICS_CODING**: Automated optimization via the built-in JavaScript terminal. Define complex behaviors for your fleet to handle peak-hour traffic.

## ⚙️ Core Systems
- **Logistics Economy**: Earn CR to upgrade Elevator Speed, Capacity, Acceleration, and Fleet Size (up to 4 elevators).
- **Sector Integrity**: Manage system load carefully; high unserved counts lead to **Network Collapse**.
- **Performance HUD**: Real-time tracking of Credits, Streak Multipliers, Average Wait Times, and Compliance ratings.
